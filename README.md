# 学知汇 · 大学版
> 轻量级校园学习资源共享平台 | 课程设计作业

## 📖 项目介绍
学知汇是一款面向大学生的轻量级学习资源共享平台，旨在解决校园内学习资料分散、获取不便的问题。
用户可以注册账号、上传各类学习资源（课件、实验报告、习题集、笔记等）、下载他人分享的资源、对资源发表评论，实现校园内学习资源的互助共享。

本项目为前后端分离架构的课程设计作业，代码简洁易读，核心功能完整，适合作为 Web 开发入门学习项目。

## 🛠️ 技术栈
### 前端
- HTML5 + CSS3（原生开发，无前端框架）
- JavaScript (ES6+)
- 原生 Fetch API 实现前后端交互
- localStorage 实现前端用户状态存储

### 后端
- Spring Boot 框架
- MyBatis 持久层框架
- RESTful API 设计规范

### 数据库
- MySQL 8.0
- InnoDB 存储引擎，utf8mb4 字符集

### 开发&构建工具
- Maven
- IntelliJ IDEA / VS Code
- Git 版本控制

## ✨ 核心功能
### 1. 用户模块
- 用户注册：用户名唯一性校验，账号密码持久化存储
- 用户登录：账号密码校验，登录状态前端本地存储
- 退出登录：清除本地登录状态，跳转回着陆页

### 2. 资源模块
- 资源上传：支持上传本地文件，自定义资源标题，自动记录上传者信息
- 资源列表：页面展示所有共享资源，包含标题、上传者、文件名
- 资源下载：点击即可下载对应资源文件
- 资源删除：仅资源上传者本人可删除自己上传的资源

### 3. 评论模块
- 发表评论：登录用户可对任意资源发表评论
- 评论列表：展示对应资源下的所有评论，包含评论者、评论内容
- 评论删除：仅评论发布者本人可删除自己发表的评论

### 4. 权限控制
- 前端级权限校验：未登录用户隐藏上传、删除等操作按钮
- 后端级权限校验：删除操作校验操作者ID与资源/评论所属用户ID是否一致

## 🚀 本地运行教程
### 环境准备
确保本地已安装以下环境：
- JDK 17+
- Maven 3.6+
- MySQL 8.0+

### 1. 数据库初始化
1.  创建数据库 `xuezhihui`
2.  执行后端项目中 `src/main/resources/sql/schema.sql` 脚本，自动创建所有数据表

### 2. 后端启动
1.  用 IntelliJ IDEA 打开 `xuezhihui-backend` 项目
2.  修改 `application.yml` 中的数据库配置，改为你本地的数据库账号密码
3.  启动 Spring Boot 项目，默认运行在 `8080` 端口

### 3. 前端启动
1.  用 VS Code 打开 `xuezhihui-frontend` 目录
2.  直接用浏览器打开 `index.html`，或使用 Live Server 插件启动即可
3.  前端默认请求后端地址：`http://localhost:8080`
